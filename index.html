Diurnyx

Modular biosignal acquisition for experimental sleep research

Diurnyx is a configurable, standalone platform for recording biosignals in sleep-related research and engineering contexts. It is intended for use in experimental environments where flexible sensor configurations and reproducible data collection workflows are required.

Documentation

Purpose

Diurnyx is designed to support exploratory and experimental work involving biosignal acquisition in sleep-related contexts.

It provides a hardware platform where sensor configurations can be defined in software, allowing measurement setups to be adjusted between experiments without modification of the underlying system.

The system is intended to facilitate iterative development, prototyping, and data collection for studies involving sleep and related physiological signals in non-clinical settings.

System Characteristics

  • Configurable sensor architecture: Sensor selection and configuration are defined via a configuration file, enabling repeatable and adjustable measurement setups across different sleep-related experiments.
  • Compact, wearable form factor: Lightweight and practical for extended recording sessions, including overnight studies, with consideration for wearability during data collection.
  • Standalone operation: No host computer required during recording. Powered via USB-C and records directly to removable storage.
  • Standardized data format: Data is stored in EDF (European Data Format), commonly used in sleep research workflows.
  • Deterministic data handling: Each recording session is saved as a separate, timestamped file on a FAT-formatted SD card.

Sensors

Diurnyx currently supports a set of sensors commonly used in exploratory sleep and physiological signal acquisition. Sensor configurations are defined in software and can be adjusted between experiments.

  • Pressure transducer: For airflow measurement applications.
  • Temperature: High-precision temperature measurement (±0.1°C).
  • Triaxial accelerometer: Motion detection and orientation.

Currently under development:

  • Respiratory Inductance Plethysmography (RIP): Respiratory effort and breathing patterns.
  • Photoplethysmography (PPG): Heart rate and peripheral oxygen saturation.

Additional sensor types may be supported in future revisions as the platform evolves.

Technical Summary

  • Modular sensor support with configuration via config.json
  • SD card storage (FAT filesystem)
  • Timestamp-based session files
  • EDF (European Data Format) output
  • Firmware implemented in Rust for system stability
